Creating New Blog Posts

To add a new blog post:

  1. Create a new HTML file (e.g., my-new-post.html) in the root directory.
  2. Follow the structure shown in example-post.html for formatting questions, AI responses, and code.
  3. Link the new HTML file from this main page (index.html) by adding a new
    section under "Recent Posts".
  4. (Optional) Update the category list if needed.

This system allows for easy creation and linking of individual post pages.
